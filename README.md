# Arcade

오락실을 다니면서 위치를 찾기 어려워 만든 iOS 앱입니다.  
현재 위치를 기준으로 가까운 오락실을 지도와 목록으로 빠르게 확인할 수 있습니다.

## 주요 기능
- 현재 위치 기반 오락실 거리 정렬
- 지도 핀 + 목록 연동
- 상세 시트(주소/메모/길찾기)

## 사용 스택/기술
- UIKit
- MapKit
- CoreLocation
- Storyboard

## 앱 구조
- 메인 화면: 지도 + 목록

## 데이터 소스
- `arcade.json` 로컬 데이터 활용

## 실행 방법
- Xcode에서 빌드/실행 → 위치 권한 허용 → 목록 선택 → 상세 → “길찾기”
